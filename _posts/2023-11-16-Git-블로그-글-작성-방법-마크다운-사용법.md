## Git 블로그 글 작성 방법

_posts 폴더에 YYYY-MM-DD-제목.md 파일 생성후 포스트 front matter를 아래와 같이 작성하시면 됩니다.
```---
title: "제목"
excerpt: ""

categories:
  - 카테고리제목
tags:
  - []

permalink: /카테고리명/제목

toc: true
toc_sticky: true
 
date: YYYY-MM-DD
last_modified_at: YYYY-MM-DD
---
```

포스트에 사용할 이미지는 assets/images/posts_img/post-name-here/ 하위에 저장
