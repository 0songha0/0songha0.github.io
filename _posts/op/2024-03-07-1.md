## 압축 해제 명령어

### zip 파일 압축 해제
```
unzip 파일명.zip
또는
unzip 파일명.zip -d ./압축해제폴더명
```
경로를 지정하지 않으면 현재 경로에 모든 파일을 압축해제합니다.

### jar, war 파일 압축 해제
```
jar -xvf 파일명.war
```

### tar 파일 압축 해제
```
tar -xvf 파일명.tar -C ./압축해제폴더명
```
