## 서버 IP 및 도메인 확인

### 서버 내부 IP 확인
```
ifconfig
또는
ip addr
```
eth0:의 inet 오른쪽에 내부 IP가 나옵니다.

### 서버 외부 IP 확인
```
curl ifconfig.me
```

### ip로 도메인 확인
```
nslookup ip
```
ip에 연결된 도메인이 있는데 찾을 수 없는 경우도 있습니다.

### 도메인으로 ip 확인
```
nslookup 도메인명
```
http://를 제외하고, com 또는 co.kr까지의 도메인만 입력해야 나옵니다.
