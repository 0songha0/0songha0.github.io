## 서버 IP 및 도메인 확인

### 내부 IP 확인
```
ifconfig 또는
ip addr
```
eth0:의 inet 오른쪽에 내부 IP가 나옵니다.

### 외부 IP 확인
```
curl ifconfig.me
```
