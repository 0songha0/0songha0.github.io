## Codex AI란?
chatGPT를 제공하는 OpenAI에서 만든 코딩 어시스턴트입니다.  
작업을 단계적으로 나눠서 처리하여 Claude Code 보다 작업 속도가 느리게 느껴졌습니다.

### 윈도우 Codex 설치 방법 종류
- IntelliJ 최신버전에서 플러그인으로 설치
- exe 파일으로 Codex 앱 설치
- cmd 창에서 npm 설치 후 cli 명령어로 설치

## 윈도우 Codex 앱 설치 방법

### 윈도우 Codex 앱 다운로드
<https://openai.com/ko-KR/codex>{: target="_blank"}  
위 링크에서 Codex Installer.exe 파일을 다운로드하고 설치합니다.

### Codex 로그인
Codex 앱 초기 실행 시, GPT 계정 또는 openAI API 키로 로그인합니다.

### 프로젝트 추가
새 프로젝트 추가 > 기존 프로젝트 폴더 경로 이동 > 폴더 선택

### 샌드박스 권한 설정
명령 프롬프트 창 위 샌드박스 허용 팝업에서 Set up 버튼 클릭
