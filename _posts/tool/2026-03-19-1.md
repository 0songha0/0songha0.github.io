## Codex AI란?
chatGPT를 제공하는 OpenAI에서 만든 코딩 어시스턴트입니다.

### Codex AI 유료 사용 후기
작업을 단계적으로 나눠서 처리하여 Claude Code 보다 체감 작업 속도가 느렸습니다.

정렬 순서를 SQL이 아니라 Java 서비스에서 변경해 비효율적으로 느껴졌고,  
복잡한 정렬 기준을 모두 반영하지 않아서 결과 순서가 뒤죽박죽으로 나왔습니다.

컨트롤러에서 범용적으로 사용 중인 객체의 import 문을 제거하여 오류가 발생하기도 하였습니다.

같은 요청을 Claude에 하니까 신속하고 정확하게 처리해 주는 모습을 보였습니다.  
Codex AI가 좀 더 발전하면 다시 사용해 봐야겠다는 생각이 들었습니다.

### 윈도우 Codex 설치 방법 종류
- IntelliJ 최신버전에서 플러그인으로 설치
- exe 파일으로 Codex 앱 설치
- cmd 창에서 npm 설치 후 cli 명령어로 설치

## 윈도우 Codex 앱 설치 방법

### 윈도우 Codex 앱 다운로드
<https://openai.com/ko-KR/codex>{: target="_blank"}  
위 링크에서 Codex Installer.exe 파일을 다운로드하고 설치합니다.

### Codex 로그인
Codex 앱 초기 실행 시, GPT 계정 또는 openAI API 키로 로그인합니다.

### 프로젝트 추가
새 프로젝트 추가 > 기존 프로젝트 폴더 경로 이동 > 폴더 선택

### 샌드박스 권한 설정
명령 프롬프트 창 위 샌드박스 허용 팝업에서 Set up 버튼 클릭
