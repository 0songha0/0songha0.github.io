## 설정한 환경변수 확인
```
echo %환경변수명%
```

### 현재 디렉토리 파일 및 폴더 확인
```
dir
```

### 내부 IP 확인
```
ipconfig
```
DNS 서버 등 더 상세한 네트워크 정보 확인은 아래 명령어로 가능합니다.
```
ipconfig -all
```

### D드라이브 이동
```
D:
```

### 도메인으로 ip 확인
```
nslookup 도메인
```

### 맥 주소 확인 (MAC Address)
```
ipconfig -all
```
이더넷 어댑터 이더넷:의 물리적 주소가 맥 주소입니다.
```
getmac -v
```
이더넷의 물리적 주소가 맥 주소입니다.
