## VSCode 사용법

### 프로젝트 여는 방법
상단 File 메뉴 > Open Folder > 프로젝트 폴더 선택

### 파일 내 텍스트 검색
좌측 검색 아이콘 클릭 > 검색어 입력

### 플러그인 설치 방법
좌측 EXTENSIONS > 플러그인명 검색 > 플러그인 선택 > Install  
파이썬 플러그인도 설치할 수 있습니다.
