## FileZilla 사용법

### 개발서버 SFTP 연결 방법
파일 > 사이트 관리자 > New site > 이름 바꾸기 > 서버명 입력 후 우측에 아래와 같이 서버 정보를 입력하고 연결합니다.
<table class="table_2_left">
  <tbody>
    <tr>
      <td>프로토콜(t)</td>
      <td>SFTP - SSH File Transfer Protocol 선택</td>
    </tr>
    <tr>
      <td>호스트(H)</td>
      <td>개발서버IP 입력</td>
    </tr>
    <tr>
      <td>포트(P)</td>
      <td>22 입력</td>
    </tr>
    <tr>
      <td>로그온 유형(L)</td>
      <td>키 파일 선택</td>
    </tr>
    <tr>
      <td>사용자(U)</td>
      <td>개발서버 리눅스 계정 유저명 입력</td>
    </tr>
    <tr>
      <td>키 파일(K)</td>
      <td>찾아보기 선택 후 개발서버 인증서 (프로젝트명-dev.ppk) 삽입</td>
    </tr>
  </tbody>
</table>
