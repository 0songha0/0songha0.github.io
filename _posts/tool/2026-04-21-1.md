## Claude DB 연결 방법

### Claude Desktop 설정 파일 수정
파일 > 설정 > 개발자 > 로컬 MCP 서버 구성 편집 > claude_desktop_config.json 파일 열기
```
{
  "preferences": {
    "coworkWebSearchEnabled": true,
    "coworkScheduledTasksEnabled": false,
    "ccdScheduledTasksEnabled": false
  },
  "mcpServers": {
    "genia-mysql": {
      "command": "npx",
      "args": ["-y", "mysql-mcp-server"],
      "env": {
        "MYSQL_HOST": "10.10.101.DB서버IP",
        "MYSQL_PORT": "3306",
        "MYSQL_USER": "유저명",
        "MYSQL_PASSWORD": "비밀번호",
        "MYSQL_DB": "DB명",
        "MYSQL_TIMEZONE": "+09:00"
      }
    }
  }
}
```
위와 같이, 기본 설정 아래에 mcp 서버 및 DB 연결 정보를 추가하고 저장합니다.  
작업 관리자에서 Claude 완전 종료 후 재시작 시 로컬 MCP 서버가 확인되며, DB 연결이 가능해집니다.
