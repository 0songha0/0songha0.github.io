## 다크모드 설정 방법

### 윈도우 다크모드 설정
Windows 설정 > 개인 설정 > 색 > 모드 선택 : 다크 선택  
윈도우에서 관리하는 모든 웹 브라우저, 파일탐색기, 제어판 등에 다크모드가 적용됩니다.

### 크롬 다크모드 설정
크롬 우측 상단 점 3개 > 설정 > 모양 > 모드 > 어둡게 선택  
크롬 상단바, 어두워져도 괜찮은 화면들만 어두워집니다.

### 포스트맨 다크모드 설정
상단 설정 아이콘 > Settings > Themes > Dark 선택

## 라이트모드 설정 방법

### 윈도우 라이트모드 설정
Windows 설정 > 개인 설정 > 색 > 모드 선택 : 사용자 지정 선택 > 기본 Windows 모드 선택 : 다크 (작업표시줄 검은색 설정) > 기본 앱 모드 선택 : 라이트

### 크롬 다크모드 설정 해제
```
chrome://flags/#enable-force-dark
```
다크모드 시 어두워지면 안 되는 화면도 어두운 경우, 크롬 주소창에 위와 같이 입력합니다.  
Auto Dark Mode for Web Contents 항목 Disabled 후 다시 시작하면 해결됩니다.

### 포스트맨 라이트모드 설정
상단 설정 아이콘 > Settings > Themes > Light 선택

### VSCODE 라이트모드 설정
좌측 상단 메뉴 > File > Preferences > Settings > theme 검색 > Workbench: Color Theme : Solarized Light 선택

### IntelliJ 라이트모드 설정
좌측 상단 File > Settings > Appearance & Behavior > Appearance > Theme: IntelliJ Light 선택 > OK

### 이클립스 라이트모드 설정
Window > Preferences > General > Appearance > Theme : Light 선택

### 노트패드 라이트모드 설정
상단 설정 > 환경설정 > 다크모드 > 다크모드 활성화 체크 해제 > 닫기

### PuTTY 라이트모드 설정
Session 설정 선택 > Load > Window > Colours > Select a colour to adjust > Default Background (배경색) : 255, 255, 255 입력 (흰색) > Default Foreground : 0, 0, 0 입력 (검은색) > Session 설정 선택 > Save > Open  
저장된 PuTTY Session 설정은 MTPuTTY 툴로 세션에 접속해도 자동 적용됩니다.
