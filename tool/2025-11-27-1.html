    윈도우 CLI claude 설치 방법

    윈도우 PowerShell 실행

    윈도우 검색 > PowerShell 입력 > Windows PowerShell 실행
    cmd에서는 질문할 때 여러 줄 불여넣기가 되지 않아 PowerShell로 하는 것이 좋습니다.

    node 설치 확인

    node -v
npm -v

    Window에서 Claude AI를 사용하기 위해서는 node가 설치되어야 합니다.

    claude 설치 명령어

    npm install -g @anthropic-ai/claude-code

    node가 설치되어 있으면 위 명령어로 claude 패키지를 설치할 수 있습니다.

    claude 실행 명령어

    cd 프로젝트경로
claude

    claude 화면이 나오면 엔터를 두 번 입력하여 로그인 페이지로 진입할 수 있습니다.
    Claude 이메일 계정 입력 후 인증 코드로 인증하고 승인합니다.
    다시 cli로 이동하고 엔터를 두 번 입력하여 현재 폴더를 workspace로 사용합니다.

    PowerShell에서 claude 실행 시 오류 해결

    claude : 이 시스템에서 스크립트를 실행할 수 없으므로 D:\nodejs\claude.ps1 파일을 로드할 수 없습니다. 자세한 내용은 abou
t_Execution_Policies(https://go.microsoft.com/fwlink/?LinkID=135170)를 참조하십시오.
위치 줄:1 문자:1
+ claude
+ ~~~~~~
    + CategoryInfo          : 보안 오류: (:) [], PSSecurityException
    + FullyQualifiedErrorId : UnauthorizedAccess

    위와 같은 오류가 나오면, 아래 명령어로 현재 사용자가 ps1 스크립트를 실행할 수 있도록 허용합니다.

    Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

    claude code 초기화

    /init

    프로젝트 경로에서 실행한 claude 화면에 위 명령어를 입력합니다.
    Claude code가 프로젝트를 분석해서 만드는 CLAUDE.md 파일이 생기고, 이후 답변 정확도가 높아집니다.
    진행 과정에서 Yes, allow reading from d\ from this project, yes, and don’t ask again for similar commands in 프로젝트경로 등을 선택하면 파일 수정 시 접근 허용을 다시 요청하지 않아 편리합니다.

    윈도우 Claude CLI 사용법

    한국어 답변 설정

    CLAUDE.md 파일에 항상 한국어로 답변해달라는 내용을 추가해줘

    앞으로 항상 claude가 한국어로 답변하게 되고, 다시 영어로 답변을 변경할 수도 있습니다.
    claude는 항상 CLAUDE.md 파일을 참고하기 때문에 내용 추가를 요청하였습니다.

    CLAUDE.md 파일에 추가하면 좋은 내용

    CLAUDE.md 파일에 아래 코딩 규칙을 중복 없이 추가해줘
- 하드코딩을 지양하고, 재사용 가능하도록 공통화 및 설정화를 우선한다.
- 중복을 제거하고, 성능과 유지보수를 고려한 효율적인 코드로 개발한다.
- 최신 문법을 사용하며, 공식 문서에서 권장하는 코드 스타일을 준수한다.
- 가독성을 최우선으로 하여 불필요한 코드와 복잡도를 제거한다.
- Controller는 요청/응답 처리와 Service 호출만 담당한다.
- 모든 비즈니스 로직은 Service 계층에 작성한다.
- 서브쿼리 사용을 지양하고, 가능하면 JOIN 기반으로 쿼리를 작성한다.

    문자열 상수 등을 코드에 직접 작성하지 않고, 설정 파일로 분리합니다.
    신규 로직을 작성하기 전에 기존 유틸, 공동 함수 등을 먼저 확인하고 재사용합니다.
    함수는 한 가지 일반 수행하고, 의미 있는 변수명과 함수명을 사용하고, 중복 코드 및 불필요한 연산을 피합니다.

    프로젝트 파일 지정

    @파일명

    위와 같이 @파일명 또는 @경로/파일명을 지정하여 질문하면 좋습니다.
    프로젝트 폴더 내에 docs 폴더 생성 후 DDL, 화면설계서 등을 넣고 질문 시 참조할 수 있습니다.
    화면설계서는 파워포인트 형식은 읽을 수 없고 용량이 크면 비용이 많이 나가므로,
    필요한 특정 페이지만 추출하여 PDF로 만들어서 넣는 것이 좋습니다.

    파워포인트 PDF 변환 방법
    파워포인트 화면 > 파일 > 내보내기 > PDF/XPS 만들기 > 경로 선택 > 게시

    가이드 파일 요청

    ~ 기능을 구현하고 싶은데 어떻게 개발할지 가이드 파일을 먼저 만들어줘

    AI가 제시한 가이드 파일을 확인하고 문제가 없으면 그대로 작업해달라고 할 수 있습니다.

    화면 개발 요청

    /user/evlTrgt/selectListProf.do 기존 선생님 평가 별 리포트 화면인데, 이 화면 코드들 참고해서
@src/main/webapp/static/html/t_map_005.html 이 퍼블리싱 파일 코드 기반으로 @docs/선생님 리포트 목록 화면.pdf 화면설계서 내용대로 선생님 리포트 목록 /user/lrn/profRprt/profRprtList.do 개발해줘

    예시처럼 참고할 기존 화면 URL, 퍼블리싱 파일, 화면설계서 파일, 신규 화면 URL 등을 명시해주면 좋습니다.

    Claude 에러 대응

    사용량 한도 초과 메시지

    Limit reached · resets 1pm (Asia/Seoul) · /upgrade to Max 20x or turn on /extra-usage

    Stop and wait for limit to reset 선택지를 선택하고, 한도가 초기화되는 1시까지 기다리면 됩니다.
    Claude는 사용량이 초과되면 미리 정해진 리셋 시각에 사용 한도가 복구됩니다.
